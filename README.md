# clint-sccaner
It is a scanner for CORS, CRLF, XSS vulnerabilities

What vulnerabilities does it scan for:
```
Cors
null cors
crlf
xss reflected
xss stored
```

In addition, this tool allows us to send GET and POST requests with arbitrary values
